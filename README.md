# WotR - Voice-Ports - Kingmaker-Companions
## About

This is an audio related mod for Pathfinder: Wrath of the Righteous.

It adds the voice barks from Pathfinder: Kingmaker's companions for use in WotR.

No vanilla voice sets are being replaced, the new ones will be added to the list.

Included are the following:
- Amiri
- Cephal Lorentus
- Ekundayo
- Harrim
- Jaethal
- Jubilost
- Kalikke
- Kanerah
- Linzi
- Maegar Varn
- Nok-Nok
- Octavia
- Regongar
- Tartuccio
- Tartuk
- Tristian
- Valerie

## Installation
Download and install [Unity Mod Manager](https://www.nexusmods.com/site/mods/21).

Download the zip file from the latest release of this mod from [the releases](https://github.com/MrParadux/WotR_Voice-Ports_Kingmaker-Companions/releases)

Drag the mod's zip file in the Unity Mod Manager's "Mod" tab or select it manuall with "Install Mod".

## Additional Notes
If you encounter any issues like low or high volume, a lack of barks, too many repetetions or encounter other problems, please report them as a bug in the [issues section](https://github.com/MrParadux/WotR_Voice-Ports_Kingmaker-Companions/issues) so that I can address them.

Similarly, if you have ideas or suggestions, please also post them in the issues section as an enhancement.
